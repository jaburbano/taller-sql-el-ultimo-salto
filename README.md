## Descripción del proyecto

Este repositorio contiene la solución completa del taller SQL basado en la base de datos **tienda.db**, correspondiente al caso de negocio "El Último Salto".

El taller incluye consultas SQL desde nivel básico hasta avanzado, utilizando:

- SELECT
- WHERE
- ORDER BY
- JOIN
- GROUP BY
- HAVING
- funciones agregadas (SUM, COUNT, AVG, MAX, MIN)
- subconsultas
- CTE (WITH)
- análisis de datos

Cada archivo `.sql` corresponde a la solución de un ejercicio específico.


## Herramientas utilizadas

- SQL Online
- GitHub

## Cómo ejecutar

1. Abrir la base de datos `tienda.db` en SQLite
2. Ejecutar cualquier archivo `.sql`
3. Ver los resultados
